# How to build (Linux)
## Step 1: Install Dependencies
Use apt, pacman, or whatever your distro allows (package names way vary).
```
g++ git cmake qt6-base-dev
```
## Step 2: Build
```
git clone https://github.com/BURNlNgWOLF/diskra2.0.git
cd diskra2.0
mkdir build && cd build
cmake ..
cmake --build .
./Diskra
```
## Step 3: Enjoy
